The Problem

Misdiagnosis costs harvests.

Smallholder farmers frequently misidentify plant diseases, leading to wrong or delayed treatment and avoidable crop loss. Expert agronomists are not always reachable in time.

LeafGuard puts a trained image-classification model behind a simple web interface: a farmer photographs an affected leaf, and the system returns the likely disease, a confidence score, severity, and an actionable treatment recommendation — with results logged and mapped to surface local outbreaks.

Model Pipeline

From photo to recommendation

A four-stage inference flow — order matters, so each stage feeds the next.

01

Capture

User submits a leaf image via upload or live camera.

02

Preprocess

Image resized and normalised to the model's input format.

03

Classify

Fine-tuned CNN predicts disease class with a confidence score.

04

Advise

Result mapped to treatment guidance, logged and mapped.
